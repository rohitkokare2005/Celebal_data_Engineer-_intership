 Project Overview

This project demonstrates SQL-based sales data analysis using **MySQL Workbench**. The objective is to analyze an e-commerce sales database by applying SQL concepts such as filtering, aggregation, joins, sorting, indexing, CASE statements, transactions, and business intelligence queries.

The project follows the Week 2 assignment requirements provided by Celebal Technologies.
